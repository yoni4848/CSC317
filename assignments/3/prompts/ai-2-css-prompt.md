# AI-2 CSS Prompt (Image + Text)

**Image Reference:** images/ai-2-reference.png  

**Goal:** Create a sleek, professional, dark-mode portfolio theme inspired by brittanychiang.com.  
Match the exact structure of Yoni Melkie’s `index.html` while incorporating a modern aesthetic with clean typography, smooth animations, and high contrast between text and background.

**HTML Structure to Match:**
- Header and Navbar: `<header>` and `.headnav`, `.logo`, `nav a`
- Hero image: `#profile_pic`
- About section: `#About`
- Skills section: `#Skills`
- Projects section: `#Projects`
- Contact / Footer: `#Contact`, `.icons`
- Theme switcher: `.theme-switcher`, `.theme-buttons`, `.theme-button`, `.active`

**Design Requirements:**
- **Color Palette:** Dark navy background (#0A192F), soft white text (#CCD6F6), bright accent green (#64FFDA)
- **Typography:** Clean sans-serif (“Poppins” or “Inter”), larger heading weight
- **Header:** Sticky with transparent background and underline hover animation on nav links
- **Hero:** Centered image with name and subtitle; large, bold name in accent color
- **About/Skills:** Two-column layout with readable spacing; soft fade-in effect
- **Projects:** Grid or card layout with glowing border hover animation
- **Contact:** Centered icons with accent-color hover glow
- **Buttons:** Rounded edges, smooth color transitions, light accent hover
- **Animations:** Soft fade-in and slide-up transitions for sections
- **Responsive:** Fully responsive across mobile, tablet, and desktop
- **Variables:** Define :root colors for --background, --text, and --accent
- **Comments:** Add comments explaining each main section
- **Output:** Return **only CSS code**, no HTML or JS
