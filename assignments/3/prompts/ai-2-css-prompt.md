# AI-2 CSS Prompt (Image + Text)

**Image Reference:** images/ai-2-reference.png  

**Goal:** Generate a modern fintech-style CSS stylesheet for Yoni Melkie’s portfolio website, inspired by the Binance.US homepage aesthetic.

**Prompt:**
You are a professional front-end web designer. Using the attached Binance.US reference image, generate a complete CSS stylesheet that matches the structure of Yoni Melkie’s `index.html` file. Use the same section and class names from that HTML. Do not invent new IDs or class names.

**Design Requirements:**
- **Color Palette:** White background (#FFFFFF), dark gray text (#1E1E1E), and yellow accent (#F0B90B)
- **Typography:** Bold, modern sans-serif fonts (e.g., Inter, Helvetica, or Poppins)
- **Layout:** Clean grid/flexbox layout with large hero heading, clear spacing, and balanced sections
- **Header/Nav:** Sticky header with hover and focus states on navigation links
- **Buttons:** Rounded corners, yellow background with black text, smooth hover transitions
- **Sections:** Style About, Skills, Projects, and Contact to have consistent padding and alignment
- **Images:** Responsive with soft drop shadows
- **Responsiveness:** Fully responsive (mobile, tablet, and desktop)
- **Animation:** Add one subtle fade-in animation on section scroll
- **Variables:** Use CSS variables for colors, fonts, and spacing
- **Comments:** Add short comments explaining each major section
- **Output:** Return only CSS code — no HTML or JavaScript.
